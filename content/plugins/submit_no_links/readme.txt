Submit No Links Plugin for Hotaru CMS
---------------------------------------------
Created by: Nick Ramsay

Description
-----------
This plugin removes the first step of submission from the Submit plugin and makes all posts "editorials" so you don't need to submit a link.

Instructions
------------
1. Ensure that the original Submit plugin is installed and active.
2. Upload the "submit_no_links" folder to your plugins folder. 
3. Install it from Plugin Management in Admin.
